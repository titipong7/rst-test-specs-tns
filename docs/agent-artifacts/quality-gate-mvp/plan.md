# Pilot Plan: Quality Gate Workflow

## Scope

- Add reusable multi-agent workflow assets for Cursor.
- Add baseline rules and skill templates.
- Add quality-gate command and docs.

## Non-Goals

- No changes to production spec behavior.
- No CI pipeline refactor in this pilot.

## Acceptance Criteria

1. Role definitions and handoff contracts exist in docs.
2. Cursor rules and skills are present for Planner/Builder/Reviewer/Tester.
3. Quality gates are documented and executable.
4. Pilot report includes measured cycle time and error profile.
